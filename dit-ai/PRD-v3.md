## 一、产品定位

**Dit.ai 是全球AI算力流通协议。**

全球每年有数十亿美元的AI订阅额度被白白浪费——Anthropic Max、Kiro、Antigravity、OpenAI Pro的用户，大量额度每月清零。Dit让这些算力流动起来，成为可交易的商品。

Dit不是撮合平台，不是API代理。Dit是**AI算力的全球做市商**——供应商报底价，买家付顶价，平台掌握定价权，吃中间spread。

每一个OpenClaw实例天然是Dit的一个节点——既可以作为供应商贡献闲置算力，也可以作为买家以最优价格消费算力。目标：**年底前接入30万台OpenClaw实例**，构建全球最大的去中心化AI算力网络。

## 三、系统架构

```
OpenClaw实例（买家模式）
  │
  ▼
Dit Gateway（统一API入口）
  │  ├── 鉴权（实例API key）
  │  ├── 计费（扣买家余额）
  │  ├── 路由决策（全网最优供应商）
  │  └── 请求转发
  │
  ▼
OpenClaw实例（供应商模式，内置Dit Adapter）
  │  ├── 接收Gateway路由请求
  │  ├── 用自己的API key和IP发起请求
  │  └── 返回结果给Gateway
  │
  ▼
官方API（Anthropic / OpenAI / Kiro / Antigravity...）
```

**关键原则：**
- Dit平台永远不持有供应商的API key
- 所有对官方的请求，从供应商自己的IP发出
- Dit只是路由层，法律风险隔离清晰
- 每个OpenClaw实例可同时是供应商和买家

## 六、资金流与浮存金价值

```
买家预充值 → Dit账户（预付款，立即到账）
     ↓
Dit Gateway计费（按调用实时扣减）
     ↓
供应商月结（后结算，T+20）
```

**年底浮存金 ~$48M 的战略价值：**
- 无成本持有，相当于$48M的零息贷款
- 可用于短期理财，额外产生2-3%年化收益（≈$1M+/年）
- 随规模增长，浮存金持续扩大，形成准金融资产

## 七、增长路径

### 阶段一：点火（2026 Q1，0→2,000节点）
- MyClaw Dashboard一键接入Dit
- 早期节点额外激励：首月分成+10%
- 目标：验证产品跑通，首批收入落地

### 阶段二：飞轮（2026 Q2，2,000→25,000节点）
- 新用户注册流程内嵌Dit接入引导
- 公开节点收入数据（脱敏排行榜）→ FOMO效应
- 开发者文档+开源Adapter → 外部实例接入

### 阶段三：爆发（2026 Q3~Q4，25,000→300,000节点）
- Dit接入成为OpenClaw实例默认功能（opt-out）
- 企业级节点批量接入
- MyClaw用户裂变：推荐新节点接入，获得其首月流水3%

### 节点激励设计
```
基础收入：实际交易额 × 分成比例（扣除Dit spread后）
稳定性奖励：月在线率 > 99% → 额外+3%
推荐奖励：推荐新节点 → 永久获得其流水的1%
早鸟奖励：前10,000个节点 → 终身+5%分成
```
