## Dit.ai 新业务讨论

Leo提出做**Dit.ai**——全球AI算力交换网络，OpenClaw生态的一部分。

### 核心设计决策（本次对话确定）

**定位：** 全球AI算力流通协议，做市商模型（不是撮合平台）
- 供应商报底价，买家付顶价，Dit掌握黑箱定价权，吃spread
- 类比：外汇/股票做市商

**资金流：** 买家预付款 + 供应商后结算 → 永远净现金正向，浮存金随规模增长

**Dit vs MyClaw关系：** B选项——Dit是独立产品，MyClaw是第一个锚定客户，独立盈利不融资

**系统架构（三层）：**
1. Dit Gateway — 统一API入口（OpenAI格式兼容），路由+计费+鉴权
2. Dit Adapter — 开源，供应商部署在自己服务器（API key不离境，从供应商IP发请求）
3. Dashboard — 供应商报价管理 + 买家账户

**关键合规设计：** 平台只做路由，所有官方API请求从供应商自己服务器发出 → 法律风险隔离

**供给侧洞察：** MyClaw用户（有OpenClaw实例+Anthropic Max/Kiro/Antigravity订阅）是天然第一批供应商；通过Dit变现可抵扣MyClaw订阅费 → 强留存

**冷启动：** MyClaw 3000+用户双边激活（既是供应商又是买家）
