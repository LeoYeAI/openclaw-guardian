## openclaw-backup skill 完成（2026-03-02）

### 背景
Leo 要求：定期一键保存 OpenClaw 所有配置+记忆+skills，支持还原到新实例。

### 构建内容
- `scripts/backup.sh` — 全量备份（workspace/config/skills/cron/scripts/identity）
- `scripts/restore.sh` — 还原（支持 --dry-run 预览，还原前自动保存快照）
- `scripts/schedule.sh` — 系统 cron 定时备份配置
- `references/what-gets-saved.md` — 备份内容说明文档
- 已打包：`~/.openclaw/workspace/skills/openclaw-backup.skill`
- 已 commit：git commit fcda41f

### 关键设计决策
- 备份格式：tar.gz（单文件，带时间戳，自动保留最近7个）
- **不备份**：credentials/auth tokens（安全）、日志、二进制媒体
- 还原前自动保存 pre-restore 快照
- 迁移流程：备份 → scp → restore.sh --dry-run → 执行 → 重新 pair channels

### 自动化
- 已添加 OpenClaw cron job（ID: 08d72872）：每日 UTC 03:00 自动备份
- 备份输出目录：/tmp/openclaw-backups/

### rsync 依赖
- 容器内原本没有 rsync，已 apt-get install -y rsync
