OpenClaw Guardian 守护进程

OpenClaw 配置保护方案 - Guardian 守护进程

概述

=== 概述 ===

Guardian 是一个独立运行的守护进程，用于监控 OpenClaw Gateway 状态。
当 Gateway 启动失败时，自动尝试修复或回滚，确保 AI 助手 24h 稳定运行。

功能特性

=== 功能特性 ===
  • 自动监控 - 每 30 秒检测 Gateway 状态
  • 自动修复 - 启动失败先尝试 doctor --fix
  • 自动回滚 - 修复失败自动回滚到上一稳定版本
  • 自动通知 - 通过 Discord API 发送告警通知

使用方法

=== 使用方法 ===
1
初始化 Git 仓库
bash
cd ~/.openclaw/workspace
git init && git add -A && git commit -m "initial"
2
创建脚本
bash
mkdir -p ~/.local/bin
vim ~/.local/bin/guardian.sh
chmod +x ~/.local/bin/guardian.sh
3
配置 systemd
bash
mkdir -p ~/.config/systemd/user
vim ~/.config/systemd/user/openclaw-guardian.service
4
启动服务
bash
systemctl --user daemon-reload
systemctl --user enable --now openclaw-guardian.service

systemd 配置

=== systemd 配置 ===
openclaw-guardian.service
[Unit]
Description=OpenClaw Guardian

[Service]
ExecStart=/path/to/guardian.sh
Restart=always

[Install]
WantedBy=default.target

核心代码

=== guardian.sh 核心代码 ===
guardian.sh（完整）
#!/bin/bash
# guardian.sh - OpenClaw Guardian 守护进程

WORKSPACE="$HOME/.openclaw/workspace"
LOG_FILE="/tmp/openclaw-guardian.log"
CHECK_INTERVAL=30          # 检测间隔(秒)
MAX_RESTART_ATTEMPTS=3   # 连续失败最大次数
COOLDOWN_PERIOD=300       # 失败后冷却期(秒)
OPENCLAW_CMD="openclaw"

# 检查 Gateway 是否运行
is_gateway_running() {
    if systemctl is-active --user openclaw-gateway >/dev/null 2>&1; then
        return 0
    fi
    if pgrep -f "openclaw.*gateway" >/dev/null 2>&1; then
        return 0
    fi
    return 1
}

# 获取上一个稳定版本的 commit
get_stable_commit() {
    git -C "$WORKSPACE" log --all --oneline -50 | \
        grep -v -E "rollback|daily-backup|auto-backup" | \
        tail -1 | awk '{print $1}'
}

# 尝试 doctor --fix 修复
try_doctor_fix() {
    $OPENCLAW_CMD doctor --fix >> "$LOG_FILE" 2>&1
}

# 执行回滚
do_rollback() {
    CURRENT_COMMIT=$(git -C "$WORKSPACE" rev-parse HEAD)
    STABLE_COMMIT=$(get_stable_commit)

    if [ -z "$STABLE_COMMIT" ]; then
        echo "无法找到稳定版本"
        return 1
    fi

    git -C "$WORKSPACE" reset --hard "$STABLE_COMMIT"
    git -C "$WORKSPACE" commit --allow-empty -m "rollback: auto rollback from $CURRENT_COMMIT"
    return 0
}
