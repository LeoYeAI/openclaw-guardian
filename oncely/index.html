01 · 产品概述与愿景

使命

消灭 AI Agent 生态中"注册 → 拿 key → 配置 → 绑卡"的无限循环。

每一个 OpenClaw 用户安装一个新 skill，都不得不重复这个流程。Oncely 的使命是让这件事只发生一次——注册一次、绑定一次，此后所有能力一键解锁。

定位

Oncely 是 AI Agent 生态的统一 API 平台与能力做市商。

  • 对终端用户：一个账户管理所有 API 能力，统一账单
  • 对开发者：发布 skill 时声明依赖，用户零配置即可使用
  • 对 API 供应商：接入 Oncely 网络，获得 AI Agent 生态的分发渠道
  • 对 MyClaw.ai：深度集成的内置能力层，提升用户体验和粘性

品牌定位

独立品牌，非 MyClaw 子品牌。原因：

  • 拥抱 OpenClaw 开源生态——开源社区需要中立的基础设施
  • MyClaw 是 Oncely 的第一大客户和最深度集成者，但不是唯一
  • 未来可服务任何 AI Agent 平台

目标用户

MyClaw 用户

9,000+ 付费用户基础。Oncely 内置预配置，credits 体系直接复用，零摩擦。

OpenClaw 开源用户

自部署的开源用户，通过 oncely login 一行命令接入。

AI 应用开发者

Skill 开发者、独立开发者。通过 Oncely 简化 API 集成，专注产品逻辑。

API 供应商

Search、TTS、翻译等 API 提供商。接入 Oncely 获得 AI Agent 生态分发。

02 · 市场分析

核心痛点

一个典型 OpenClaw 用户安装 5 个 skill，需要：

  • 注册 5 个不同平台的账号
  • 获取 5 个 API key / token
  • 绑定 1-5 张信用卡
  • 配置 5 次环境变量或配置文件
  • 管理 5 份不同的账单

这对 non-tech 用户是劝退级别的体验。即使是技术用户，这也是纯粹的时间浪费。

04 · Oncely Connect 详细设计

双层模型

Layer 1 — Auth Connect 免费

OAuth 集成，零成本，最大化装机量。

用户一键授权第三方服务，Agent 直接读写。没有 API 调用成本（或极低），Oncely 不收费。目的是用免费工具聚集用户——微信的逻辑。

首批支持：

服务能力优先级
Gmail读写邮件、搜索、标签管理P0
Google Calendar日程查询、创建、修改P0
Notion读写页面、数据库查询P0
GitHubRepo 操作、Issue、PRP1
Slack消息发送、频道管理P1
Dropbox文件读写、分享P1

Layer 2 — API Connect 付费 · 做市商

有调用成本的商业 API。供应商报底价，Oncely 自主定价，吃 spread。

核心机制：

  • 供应商报底价：API 提供商接入 Oncely 网络时设定底价
  • Oncely 自主定价：终端价格由 Oncely 制定，供应商不可见
  • 多供应商路由：同一能力（如 Search）可有多个供应商，按成本/质量/速度最优路由
  • 用户无感：终端用户不知道也不需要知道底层是谁提供的

路由策略

当同一能力有多个供应商时，Oncely Gateway 按以下优先级路由：

  1. 可用性：供应商是否在线、是否超出 rate limit
  2. 质量评分：基于历史成功率、延迟、用户反馈
  3. 成本：底价最低的优先（在质量达标的前提下）
  4. 地理位置：就近路由，降低延迟

SDK 集成方式

OpenClaw Skill Manifest

Skill 开发者在 skill.json 中声明 Oncely 依赖：

{
  "name": "my-awesome-skill",
  "oncely": {
    "auth": ["gmail", "notion"],
    "api": ["search", "tts"]
  }
}

用户安装 skill 时，Oncely Connect 自动检测并配通所需的 Auth 和 API。

接入方式

  • MyClaw 用户：自动预配置，无需任何操作
  • OpenClaw 用户：oncely login → 输入 Oncely token → 完成
  • 开发者测试：Sandbox 环境，免费额度

05 · Oncely Pay 详细设计

支付体系

MyClaw 用户

按量扣 credits，零额外操作。

  • 与 AI Token 同一张账单
  • credits 余额统一管理
  • 无需绑定额外支付方式
  • 实时扣费，余额不足时提示充值

独立用户

$5/月起包月 or 按量付费。

  • $5/月 Starter：500 次 API 调用 + 3 个 Auth 集成
  • $15/月 Pro：5,000 次 + 10 个 Auth + 优先路由
  • $49/月 Business：50,000 次 + 无限 Auth + 专属支持
  • Pay-as-you-go：无包月，纯按量

企业用户

自定义套餐

  • 专属 SLA
  • 自定义 rate limit
  • 私有部署选项
  • 批量折扣

账单管理

  • 统一 Dashboard：所有 API 调用明细一目了然
  • 多维分析：按 skill / API 类型 / 时间段切分
  • 月度账单：自动生成 PDF，邮件推送
  • 用量预警：达到 80%/100% 额度时通知
  • 限额设置：防止意外超支

开发者结算

纯做市商模式 — 无佣金抽成

Oncely 的利润来自 spread（供应商底价与终端售价的价差），不对开发者收取任何佣金。

  • 供应商/开发者报底价 → Oncely 自主制定终端价格 → 价差即利润
  • 供应商看不到终端价格，终端用户看不到底价
  • 月度结算，支持 Stripe 打款，供应商按底价 × 调用量收款
  • 最低提现额度：$25
  • 结算 Dashboard：实时查看调用量、收入、趋势
  • 高调用量供应商享受优先路由权重（激励供给侧质量竞争）

06 · Oncely for Developers 详细设计

开发者工具链

Skill 发布与依赖声明

  • Skill 发布到 ClawHub → 在 manifest 中声明 oncely.auth 和 oncely.api 依赖
  • 用户安装 skill 时自动检测 Oncely 权限，未开通则提示一键开通
  • 开发者无需处理任何支付逻辑——全部由 Oncely Pay 托管

API 文档中心

  • 所有可用 API 的交互式文档
  • 代码示例（Python / Node.js / Shell）
  • 请求/响应 Schema
  • Rate limit 和定价信息

Sandbox 测试环境

  • 每个开发者账户 → 免费测试额度（每月 1,000 次调用）
  • 独立的测试环境，不影响生产数据
  • 模拟 Auth Connect 授权流程
  • 调试日志和请求追踪

🎨 Vibe Coding IDE（Phase 3）

在线创建、测试、发布 AI skill——门槛降到零。

  • 浏览器内开发环境，无需本地安装
  • AI 辅助编码：描述需求 → 自动生成 skill 代码
  • 实时预览和测试
  • 一键发布到 ClawHub
  • 模板库：常见 skill 类型的起手模板

供应商入驻

API 提供商接入 Oncely 网络的流程：

  1. 注册：创建供应商账户，提交 API 文档
  2. 接入：配置 Oncely Adapter（标准化接口层）
  3. 报价：设定底价（Oncely 不透露给终端用户）
  4. 审核：Oncely 团队审核质量和稳定性
  5. 上线：纳入路由池，开始接收调用
  6. 结算：月度结算，按调用量 × 底价打款

07 · 账户体系与安全

双轨账户体系

MyClaw SSO

  • MyClaw 用户零摩擦登录
  • 账户自动关联
  • 共享 credits 余额
  • 统一的身份认证令牌

独立注册

  • 邮箱 + 密码注册
  • 非 MyClaw 用户的入口
  • 独立的 Oncely credits
  • 未来支持 GitHub / Google SSO

安全机制

层面措施
API Key 管理用户不接触底层供应商 key。Oncely 统一持有企业级 key，加密存储（AES-256）
OAuth Token加密存储，定期轮换，支持用户主动撤销
传输安全全链路 TLS 1.3
调用审计完整的调用日志，支持导出和查询
权限隔离每个用户的 Auth 授权和 API 配额完全隔离
合规GDPR 就绪，数据处理协议（DPA），用户数据删除权
Rate Limiting多层限流：用户级 + 全局级，防滥用

09 · 商业模型与单位经济

收入来源

①
API Spread

供应商底价 vs 终端价的差价。核心利润来源。

预估 margin：40-60%（因能力类别而异）

②
包月订阅

独立用户 $5/月起。稳定的经常性收入。

预估 ARPU：$12/月（混合）

供应商竞争红利

多供应商竞争压低底价，终端价格由 Oncely 制定，spread 持续扩大。

预估：随规模增长 margin 提升 5-10%
